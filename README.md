# jeevanrakshak
AI-powered Golden Hour Emergency System that enables fast patient triage and smart hospital routing. It collects real-time data, prioritizes critical cases, and helps ambulances reach the nearest suitable hospital, improving response time and saving lives during emergencies.

🚑 JeevanRakshak — Live Ambulance & Emergency Response System
An AI-powered emergency healthcare platform designed to reduce response time during the Golden Hour by enabling live ambulance tracking, smart hospital navigation, and real-time emergency coordination.

🌍 Problem
During medical emergencies:

Ambulances get stuck in traffic

Hospitals are not pre-informed

Patient condition is unknown beforehand

Response time increases

➡️ Critical treatment delays occur.

💡 Solution
JeevanRakshak provides:

✅ Live ambulance tracking
✅ Smart hospital navigation
✅ Real-time map visualization
✅ Fast emergency coordination dashboard

⚙️ Installation
1️⃣ Clone Repository
git clone https://github.com/YOUR_USERNAME/JeevanRakshak.git
cd JeevanRakshak
2️⃣ Install Dependencies
npm install
3️⃣ Add Google Maps API Key
Inside index.html:

<script src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY"></script>
4️⃣ Start Server
node server.js
OR

npm start
5️⃣ Open in Browser
http://localhost:3000
🌐 Deployment (Replit)
